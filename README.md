# Taiwan Holy Young - Crypto
> Author : Curious

## 開啟 Lab
如果是第一次開啟 Lab，請執行以下指令：
```bash
cd Lab
./snippet build
./snippet start
```

如果不是第一次開啟 Lab，請執行以下指令：
```bash
cd Lab
./snippet start
```

---
## 關閉 Lab
```bash
cd Lab
./snippet stop
```

---
## 設定 CTFd
在 Lab 裡每一題的資料下，都會有一個 `README.md`，按照此檔案的資料設定 CTFd 就可以了。
